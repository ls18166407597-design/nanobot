# 硬件与环境
- **设备**: MacBook Air M2 (2022)
- **屏幕分辨率**: 2560 x 1664 (Retina)

# 视觉操作偏好 (Vision Preferences)
- **一级策略 (默认)**: 优先使用 `mac_vision` 进行屏幕识别。
- **二级策略 (降级)**: `mac_vision` 无法满足任务时，改用压缩截图。
- **压缩参数**: 使用 `sips -Z 1024` 将截图宽度限制在 1024px，以节省 Token 并提升速度。
- **三级策略 (兜底)**: 仅在需要识别微小文字或精细 UI 细节时，才使用原始全量截图。
