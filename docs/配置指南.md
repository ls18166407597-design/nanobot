# Nanobot 配置指南 ⚙️

本指南介绍了如何配置您的 Nanobot 秘书，实现“内核稳定、大脑飞跃”。

## 🚀 基础配置

现在建议使用 `nanobot` CLI 工具进行配置管理：
- **查看设置**: `nanobot config list`
- **修改参数**: `nanobot config set agents.defaults.model "gemini-1.5-flash"`
- **健康检查**: `nanobot doctor`

## 📂 核心位置说明

- **内核 (Core)**: `nanobot/` 包含所有的引擎源码。
- **大脑 (Brain)**: `workspace/` 包含你的 `IDENTITY.md`, `AGENTS.md` 等。
- **配置 (Config)**: 默认位于 `.home/config.json`。

## ⚙️ 常用环境配置

通过 `NANOBOT_HOME` 环境变量可以自定义数据存放路径：
```bash
export NANOBOT_HOME="/path/to/your/config_dir"
nanobot gateway
```
